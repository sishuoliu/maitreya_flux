# 功德轮回：众生百态 - 网页版单机游戏

## 游戏简介

《功德轮回：众生百态》是一款半合作桌游的网页版实现。2-4名玩家需要在6回合内共同化解劫难、渡化苦难众生，同时竞争个人功德与智慧的积累。

## 游戏特色

- **侘寂风格UI**：简约优雅的东方美学设计
- **暗色模式**：支持明暗主题切换
- **本地存档**：使用localStorage自动保存游戏进度
- **完整规则**：忠实还原桌游v5.8规则

## 如何开始

1. 打开 `ritual.html` 文件
2. 选择玩家人数（2-4人）
3. 为每位玩家输入姓名并选择职业
4. 点击"开始游戏"

## 职业介绍

### 农夫
- **初始资源**：资粮5，功德2，慧3
- **被动技能**：劳作时额外+1资粮
- **主动技能**：分享收成（给他人资粮，双方获得功德）

### 商人
- **初始资源**：资粮8，功德2，慧1
- **被动技能**：布施时额外+2功德
- **主动技能**：慷慨宴请（全体+功德，劫难-1）

### 学者
- **初始资源**：资粮4，功德2，慧5
- **被动技能**：修行时额外+1慧
- **主动技能**：讲学传道（给他人慧，自己获得功德）

### 僧侣
- **初始资源**：资粮1，功德5，慧5
- **被动技能**：渡化时可用功德代替资粮；护法成本-1
- **主动技能**：加持祈福（加持他人下次行动收益+2）

## 游戏目标

### 团队胜利条件
- 劫难 ≤ 12
- 渡化众生 ≥ 5

### 个人胜利
团队胜利后，功德+慧总和最高者获胜

## 行动说明

每位玩家每回合有2次行动机会：

1. **劳作**：获得3资粮（农夫4资粮）
2. **修行**：获得2慧（学者3慧）
3. **布施**：消耗2资粮，获得1功德，劫难-1
4. **渡化众生**：需要慧≥5，消耗资粮渡化众生卡
5. **护法**：消耗2资粮（僧侣1资粮），获得3功德，劫难-3

## 游戏流程

每回合包含以下阶段：

1. **发愿奖励**：获得发愿卡的每回合奖励
2. **集体事件**：翻开1张集体事件卡，影响全体
3. **个人事件**：奇数回合每人抽1张个人事件卡
4. **众生阶段**：众生滞留标记+1，超时惩罚
5. **行动阶段**：每人执行2次行动
6. **结算阶段**：偶数回合消耗资粮，检查劫难

## 技术实现

- **纯HTML/CSS/JS**：无框架依赖
- **侘寂美学**：暗色模式支持
- **本地存储**：localStorage保存游戏状态
- **响应式设计**：适配不同屏幕尺寸

## 文件说明

- `ritual.html` - 主游戏文件（HTML+CSS）
- `ritual-game.js` - 游戏逻辑（JavaScript）
