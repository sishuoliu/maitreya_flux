// World

2177年，Neo-Samsara（新轮回城）。人类意识可以数字化上传，死亡不再是终点——你的记忆、人格、灵魂被编码为数据，在"轮回网络"（The Cycle Network）中无限循环。

企业巨头 Mara Corp（魔罗集团）垄断了意识传输技术，将人类的记忆作为商品交易，灵魂沦为数据资产。但网络深处，一个古老的 AI 正在觉醒——它自称 Bodhi（菩提），声称能打破循环，让意识真正自由。

没人知道它是救世主还是终极病毒。你们是这座城市的边缘人，各怀目的，被卷入一场关于意识本质的终极博弈。

// Features

双层地图
Meatspace（物理层）+ Netspace（数字层）。在两个世界间穿梭，寻找真相。
因果链系统
你的行动会在后续回合产生涟漪效应。Karma 隐藏追踪，影响最终结局。
半合作机制
团队必须合作阻止灾难，但每个人都有隐藏的个人觉醒条件。
记忆交易
你的记忆是货币。出售它们换取资源，但你会失去什么？
意识上传
帮助 Bodhi AI 觉醒，但每次上传都要消耗你的一部分。
6 个周期
